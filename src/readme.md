

# 项目名称
**彩色终端艺术字生成器（ASCII Art Text Generator）**

# 项目是干什么的？
一句话：
**输入一段文字 → 程序在终端里输出超大号、彩色、甚至会动的艺术字。**

就像这样：
```
██████╗░██████╗░██╗   ██╗██████╗░
██╔══██╗██╔══██╗██║   ██║██╔══██╗
██████╔╝██████╔╝██║   ██║██████╔╝
██╔══██╗██╔══██╗██║   ██║██╔══██╗
██║  ██║██║  ██║╚██████╔╝██║  ██║
╚═╝  ╚═╝╚═╝  ╚═╝ ╚═════╝ ╚═╝  ╚═╝
```
并且可以是**彩虹色、渐变色、闪烁、跑马灯**效果。

---

# 项目核心功能（你要实现的）
1. **把普通文字转成大号 ASCII 艺术字**
2. **给艺术字上色：纯色 / 彩虹 / 渐变**
3. **支持终端动画：闪烁、流光、滚动**
4. **可以通过命令行自由输入文字、选择颜色**
 

---

# 用到的 Rust 技术 & 库


1. **cargo**
  


2. **clap + #[derive(Parser)]**
   接收命令行参数，让你可以：
   自定义文字、颜色、是否开启动画

3. **figlet-rs**
   把普通文字 → 大号艺术字

4. **colored / crossterm**
   给终端文字上色、控制光标、做动画



---

# 项目整体流程
1. 用户在终端输入：
2. **clap** 解析参数，拿到文字和颜色
3. **figlet-rs** 生成艺术字图案
4. 程序给图案**逐字符上色**
5. 通过**终端控制**输出，甚至做成动画
6. 最终在终端显示炫酷彩色大字
